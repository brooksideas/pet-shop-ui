# Pet Shop UI

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### Run Unit Test
```
npm run unit
```

### Run End to End Test
```
npm run e2e
```
